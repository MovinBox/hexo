hexo博客

环境

hexo文档https://hexo.io/zh-cn/docs/
1、node.js(nodejs.org)
2、git(https://git-scm.com/download/win)
3、hexo，在命令提示行输入npm install -g hexo-cli安装

新建所需文件

1、初始化博客：hexo init
2、转到该博客路径：cd
3、安装：npm install

新建文章

hexo new “新建文章”

在git仓库中新建仓库

1、new repository
其中Repositoy name要跟owner一致+github.io
2、安装git部署插件：npm install –save hexo-deployer-git

设置_config.yml

找到最底部（vscode打开
deploy:
type: git
repo: https://github.com/nameWu/nameWu.github.io.git
branch: master

访问nameWu.github.io.git

1、输入 hexo clean进行清理缓存
2、输入 hexo generate重新生成database
3、成功后，输入 hexo deploy上传远程仓库。
4、访问

主题

1、找到自己喜欢的博客主题https://hexo.io/themes/，进而在git找到源码。
2、a.拷贝主题，例如：git clone https://github.com/LouisBarranqueiro/hexo-theme-tranquilpeak.git themes/tranquilpeak； b.或者直接下载，将下载的文件存到themes文件夹
3、修改_config.yml文件中的theme内容为主题名称
4、输入 hexo clean进行清理缓存
5、输入 hexo generate重新生成database
6、输入 hexo server进行本地测试，在命令框会出现一个链接，在浏览器出入此链接看本地博客是否更换主题
7、成功后，输入 hexo deploy上传远程仓库。
